Commands [default prefix: >>]

General

Command Description
help This command provides a link to this page.
commands This command provides all the commands [if you can't access this page].
status This command provides statistics on Kiwi's performance.
support This command provides a link to the 14 Studios server.
invite This command provides an invite so you can invite Kiwi to your servers!
checkperms This command provides your permission set level.
website This command provides a link to the 14 Studios website.
profile This command provides information about your Discord account.
guild This command provides information about the guild that you execute the command in.

Fun

Command Description
8ball Well, it's self-explanatory.
cat This command delivers you a veri qt cat uwu~
dog This command works the same as the cat command, but delivers a dog instead!
reddit Basically, this command delivers a post from a NON-NSFW subreddit of your choice.
coinflip This command is basically a virtual coinflip. Self-explanatory.
rps Rock, Paper, Scissors!
roast Self-explanatory. This is basically just so people can insult others. Weird, I know, right?
sentence Generates a random sentence.

Punishment & Moderation

Command Description
clear This command purges/bulk-deletes messages in chat, up to 100.
ban Self-explanatory.
kick Also self-explanatory.
unban Yeah, self-explanatory.
warn This command allows staff teams in servers to put infractions on members.
infractions This command allows staff teams in servers to track how many infractions belong to a specific user.
unpunish Remove a specific infraction from a specific user.

Settings & Utilities

Command Description
toggle This command allows you to enable/disable commands.
setlogs This command is for setting the channel for moderation and message logs.
setsuggestions This command is for setting the channel for suggestions.
prefix This command allows you to change the prefix of the bot.
settings This command shows the disabled/enabled status of each command, the prefix, and if the server is verified/in Testing Grounds.
embed This command can generate embeds for you.
remind The remind command allows you to set reminders for the future. If you need to get reminded to start a meeting, >>remind it! If you need to get reminded to add some roles, >>remind it.
poll This is basically self-explanatory. The poll command allows you to create simple yes-no polls, for things like voting or opinions.
membercount This command spits out how many people are members in the guild that the user executed .>>membercount in.

Suggestions

Command Description
suggest You know, suggest stuff. It's pretty self-explanatory!
approve This command is used for staff members of servers to approve suggestions.
consider This command is used for staff members of servers to consider suggestions.
deny This command is used for staff members of servers to deny suggestions.
